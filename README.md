# Factory Labour Manager

A simple app to manage factory work end-to-end: create a **Project**, add the **jobs**
it includes, move each job through your own **stages** (e.g. Carpentry → Polish/Paint →
Packing), assign **labour** to whole stages or to individual tasks, track progress
quantity-wise, and record delivery to the client.

**Live app:** https://muhamdmuaz.github.io/Factory-Labour-Manager/

## How it works
**Project → Jobs → Stages → Tasks → Labour**
1. Create a project (client, site, due date).
2. Add the jobs that belong to it.
3. Open a job and move it through your stages and process steps.
4. Assign a worker to a whole stage, or give individual tasks to different workers.
5. Record delivery — including any open / untracked items.

## Features
- Projects that group multiple jobs, with overall progress
- Custom stages, process steps and sub-steps (fully editable)
- Assign labour per stage **or** per individual task
- Per-worker view: clear **Pending** vs **Done** task lists
- Quantity-wise tracking — units flow through stages
- Pause/resume a job (day-count pauses too)
- Delivery with **open / untracked items** (for old items not tracked in the app)
- Team performance dashboard with rankings and analysis charts
- Backup / restore your data
- Smart defaults that learn from your most-repeated process steps
- In-app "Update app" button to always get the latest version

## 🔒 Your data is private and safe
Whoever uses this app, **their data is saved only on their own phone** (inside that phone's
browser). It is **not uploaded to any server** and no one else can see it. Each phone keeps
its own data. Please use **Settings → Save backup** every now and then to keep a copy
(and to move it to another phone if needed).
