# Factory Labour Manager

A simple app to manage factory labour and work: add workers, create jobs, move them
through your own stages (e.g. Carpentry → Polish/Paint → Packing), track progress
quantity-wise, and record delivery to the client.

**Live app:** https://muhamdmuaz.github.io/Factory-Labour-Manager/

## Features
- Add workers and assign them to each stage of a job
- Custom stages, process steps and sub-steps (fully editable)
- Quantity-wise tracking — units flow through stages
- Pause/resume a job (day-count pauses too)
- Team performance dashboard with rankings and analysis charts
- Backup / restore your data
- Smart defaults that learn from your most-repeated process steps

## 📱 Note
This app is **currently designed for iPhone**. On iPhone, open the live link in Safari →
**Share** → **Add to Home Screen** to use it like a normal app.

Your data is saved **only on your own device** (in the browser). Use **Settings → Save backup**
to keep a copy or move it to another phone.
